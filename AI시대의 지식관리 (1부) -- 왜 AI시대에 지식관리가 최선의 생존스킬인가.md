## 요약
AI 시대에 지식관리가 최선의 생존 스킬인 이유와 효과적인 지식관리 방법론을 다룬 콘텐츠

## 핵심 문제: 지식이 축적되지 않는 현실
- 매주 반복되는 회의에서 같은 이슈 논의
- 경험에서 배운 것들이 휘발됨
- **지식 → 계획 → 실행 → 피드백** 연결고리가 끊어짐
- 같은 실패의 반복

## 지식관리의 핵심 사이클
```
배우기 → 계획하기 → 실행하기 → 결과 평가 → 개선
```
- 각 단계가 연결되어야 지속적 성장 가능
- 연결고리 유지가 지식관리의 핵심

## 정보 분류 체계

### 3가지 정보 유형
1. **일정과 경험 중심 정보**
	- 특정 시점의 상황 기록
	- 패턴 발견과 경험 성찰에 유용

2. **대화와 소통 정보**
	- 향후 유사 상황에서 참고 가능
	- 실용적 가치

3. **출판물에서 습득한 정보**
	- 자신의 상황에 맞게 해석하고 적용해야 함
	- 진정한 지식으로 변환 과정 필요

### 지식 분류 기준 3가지
1. **범용성**: 좁은 범위 → 넓은 범위로 일반화
2. **출처**: 1차 지식(직접 경험) vs 2차 지식(외부 학습)
3. **확실성**: 강한 지식(검증됨) vs 약한 지식(가설 수준)

## 성공적인 지식관리 조건

### 지속가능성이 핵심
- **복리의 원칙**: 지식이 쌓이면 더 풍부한 연결 → 새로운 영감

### 3가지 필수 조건
1. **목표 서포트**
	- 실제 성과로 이어져야 함
	- 완벽한 정리보다 실용성 중요

2. **습관 친화**
	- 기존 습관에 시스템을 녹이기
	- 새로운 습관 만들기보다 효과적

3. **흥미와 동기**
	- 가시적 성과 확인 가능해야 함
	- 지속적 동기 부여 필요

## 피해야 할 3가지 안티패턴

1. **번아웃**
	- 지식관리를 별도의 일로 생각하는 실수
	- 해결책: 업무 자체에 녹여내기

2. **소화불량**
	- 너무 많은 정보 수집에만 몰두
	- 해결책: 선별적 정보 수집과 활용 중심

3. **완벽주의**
	- 완벽한 시스템 만들기에 집착
	- 해결책: 실용적이고 점진적 개선
