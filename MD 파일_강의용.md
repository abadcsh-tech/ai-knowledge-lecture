## Markdown 파일이란
Markdown(MD) 파일은 일반 텍스트로 작성된 문서를 간단한 문법으로 서식을 지정할 수 있는 마크업 언어입니다.
#### 주요 특징
- 읽기 쉬운 일반 텍스트 형식
- HTML로 쉽게 변환 가능
- 다양한 플랫폼에서 지원
- 버전 관리 시스템과 잘 호환됨
#### 기본 문법

- 제목: # H1, ## H2, ### H3
	- 제목 앞에 #의 개수로 레벨을 구분 (최대 6레벨까지)
	- 제목 뒤에는 한 줄 띄우는 것이 일반적
- 강조: **굵게**, *기울임*, ~~취소선~~
	- 굵게: **텍스트** 또는 __텍스트__
	- 기울임: *텍스트* 또는 _텍스트_
	- 굵은 기울임: ***텍스트***
- 목록: - 또는 * 사용
	- 순서 없는 목록: -, *, + 모두 사용 가능
	- 순서 있는 목록: 1. 2. 3. 형태로 번호 사용
	- 들여쓰기로 하위 목록 생성 (탭 또는 스페이스 4개)
- 링크: [텍스트](URL) 또는 [텍스트](URL "제목")
	- 참조 링크: [텍스트][참조명], 문서 하단에 [참조명]: URL 정의
- 이미지: ![대체텍스트](이미지URL)
- 코드: `인라인 코드` 또는 ```코드 블록```
	- 코드 블록에서 언어 지정: ```python, ```javascript 등
- 인용: > 인용문 내용
	- 여러 레벨 인용: >> 또는 >>> 사용
- 표: | 열1 | 열2 | 열3 |
	- 구분선: |---|---|---|
	- 정렬: |:---|:---:|---:| (좌측, 중앙, 우측 정렬)
- 수평선: --- 또는 *** 또는 ___
- 줄바꿈: 문장 끝에 스페이스 2개 또는 빈 줄
#### 사용 용도
- 문서 작성 및 정리
- README 파일
- 블로그 포스트
- 기술 문서
- 개인 노트 (Obsidian 등)
#### 장점
- 가벼운 파일 크기
- 플랫폼 독립적
- 버전 관리 용이
- 학습이 쉬움
